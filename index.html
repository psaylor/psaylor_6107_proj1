Conway's Game of Life fascinated the world by showing that complex patterns could arise out of simple rules applied to seeming chaos. The game features a 2D grid of cells, each of which lives or dies based on its interaction with its neighbors as follows:
  1. Any live cell with fewer than two live neighbors dies, as if by underpopulation.
  2. Any live cell with more than three live neighbors dies, as if by overcrowding.
  3. Any live cell with two or three live neighbors lives on to the next generation.
  4. Any dead cell with exactly three live neighbors becomes a live cell in the next generation.
In 'Separate' mode (), the board is divided by quadrants into 4 'cities' of different colors. This mode uses slightly modified rules:
  1. Any live cell with fewer than two live neighbors of the same color dies, as if by underpopulation.
  2. Any live cell with more than three live neighbors of the same color dies, as if by overcrowding.
  3. Any live cell with two or three live neighbors of the same color lives on to the next generation.
  4. Any dead cell with exactly three live neighbors becomes a live cell in the next generation as the average colso or of its neighbors.
